# C++ Exercises

Practice problems to test your C++ knowledge!

## Exercises

### 1. FizzBuzz (`fizzbuzz.cpp`)
**Difficulty:** Beginner

The classic FizzBuzz problem. Great for practicing:
- Loops
- Conditional statements
- Modulo operator
- Output formatting

### 2. Palindrome Checker (`palindrome.cpp`)
**Difficulty:** Beginner to Intermediate

Check if strings are palindromes. Learn about:
- String manipulation
- Two-pointer technique
- String algorithms
- Character handling

### 3. Calculator Class (`calculator.cpp`)
**Difficulty:** Intermediate

Build a calculator using OOP. Practice:
- Class design
- Encapsulation
- Exception handling
- State management

## How to Use

### Compile and Run Individual Exercises

```bash
# Compile
g++ -std=c++17 -Wall -o fizzbuzz fizzbuzz.cpp

# Run
./fizzbuzz
```

### Using the Makefile

From the `cpp-learning` directory:

```bash
# Compile all exercises
make

# Run all exercises
make run-exercises
```

## More Exercise Ideas

1. **Prime Number Checker**
   - Check if a number is prime
   - Generate prime numbers up to N
   - Find prime factors

2. **Temperature Converter**
   - Convert between Celsius, Fahrenheit, and Kelvin
   - Use classes for different scales

3. **Simple Bank Account**
   - Create account class
   - Implement deposit/withdraw
   - Track transaction history

4. **Sorting Algorithms**
   - Implement bubble sort
   - Implement selection sort
   - Compare performance

5. **Linked List**
   - Create a linked list class
   - Implement insert, delete, search
   - Practice with pointers

6. **File Reader**
   - Read and parse text files
   - Count words, lines, characters
   - Practice file I/O
